## C. Short description

### Επιλογή 1

Το SafeMe βοηθά να οργανώσεις έμπιστες επαφές, μήνυμα SOS και κοινοποίηση τοποθεσίας για στιγμές που χρειάζεσαι υποστήριξη.

### Επιλογή 2

Μια απλή εφαρμογή προσωπικής ασφάλειας για SOS, έμπιστες επαφές, προετοιμασμένο SMS και κοινοποίηση τοποθεσίας με δική σου ενέργεια.

### Επιλογή 3

Κράτησε τις βασικές πληροφορίες ασφάλειας σε ένα μέρος: επαφές, SOS μήνυμα, τοποθεσία, test mode και προαιρετικό συγχρονισμό λογαριασμού.

## D. Full description

Το SafeMe είναι μια εφαρμογή προσωπικής ασφάλειας που σε βοηθά να προετοιμάσεις βασικές ενέργειες υποστήριξης, χωρίς πολύπλοκες διαδικασίες.

Με το κουμπί SOS μπορείς να περάσεις σε λειτουργία έκτακτης ανάγκης μέσα από την εφαρμογή. Το SafeMe σε βοηθά να έχεις έτοιμες τις έμπιστες επαφές σου, να συνθέσεις ένα προετοιμασμένο SMS και, εφόσον δώσεις άδεια, να συμπεριλάβεις πληροφορίες τοποθεσίας στο μήνυμα ή στη ροή υποστήριξης.

Μπορείς να προσθέσεις έμπιστα άτομα, να ενημερώσεις το προφίλ σου και να χρησιμοποιήσεις test mode για να δοκιμάσεις τη ροή χωρίς να δημιουργείς άσκοπη ανησυχία. Η σύνδεση με λογαριασμό είναι προαιρετική και μπορεί να χρησιμοποιηθεί για συγχρονισμό δεδομένων όπου υποστηρίζεται.

Το SafeMe δεν αντικαθιστά τις επίσημες υπηρεσίες έκτακτης ανάγκης. Δεν εγγυάται διάσωση, ανταπόκριση ή αυτόματη επικοινωνία με επίσημες αρχές. Σε πραγματικό κίνδυνο, κάλεσε άμεσα τον τοπικό αριθμό έκτακτης ανάγκης ή επικοινώνησε με τις αρμόδιες υπηρεσίες.

Η κοινοποίηση τοποθεσίας απαιτεί άδεια από τον χρήστη και μπορεί να εξαρτάται από τη συσκευή, το δίκτυο και τις ρυθμίσεις απορρήτου. Τα μηνύματα SMS προετοιμάζονται από την εφαρμογή, αλλά η αποστολή τους γίνεται χειροκίνητα από τον χρήστη μέσα από την εφαρμογή μηνυμάτων της συσκευής.

## E. Key features

- SOS mode για γρήγορη πρόσβαση σε ροή προσωπικής ασφάλειας
- Trusted contacts / έμπιστες επαφές για άτομα που θέλεις να ειδοποιήσεις
- Location sharing με άδεια χρήστη
- Prepared SMS composer για προετοιμασμένο μήνυμα SOS
- Optional account sync για προαιρετικό συγχρονισμό λογαριασμού όπου υποστηρίζεται
- Test mode για ασφαλή δοκιμή της ροής
- Privacy and safety guidance με σαφείς σημειώσεις για άδειες, περιορισμούς και υπεύθυνη χρήση

## H. Store compliance notes

- Το SMS προετοιμάζεται από την εφαρμογή, αλλά ο χρήστης το ελέγχει και το στέλνει χειροκίνητα.
- Η τοποθεσία απαιτεί ρητή άδεια χρήστη και μπορεί να μην είναι πάντα διαθέσιμη ή ακριβής.
- Ο λογαριασμός είναι προαιρετικός και δεν απαιτείται για κάθε βασική χρήση της εφαρμογής.
- Η εφαρμογή δεν είναι επίσημη υπηρεσία έκτακτης ανάγκης και δεν επικοινωνεί αυτόματα με επίσημες αρχές.
- Υπάρχουν draft κείμενα για Privacy Policy και Terms of Use που πρέπει να ελεγχθούν πριν από δημοσίευση.
